WebSocket / OSC Integration

Websocket Server IP
Websocket Server Port

OSC can not be sent directly from the browser, that's why we send to a locally hosted Websocket server (which then sends the OSC message).

The desktop version of TTS Voice Wizard has a Websocket server built-in for receiving text from the web app.

If your server is on a different machine make sure to change the IP from localhost to the actual IP address.
